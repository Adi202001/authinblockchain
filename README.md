# Decentralized Two-Blockchain Password Manager

A Java-based decentralized password manager with two interconnected blockchains for user authentication and secure password storage.

## Table of Contents
- [Features](#features)
- [Prerequisites](#prerequisites)
- [Installation](#installation)
- [Usage](#usage)
- [Configuration](#configuration)
- [Contributing](#contributing)
- [License](#license)

## Features

- User registration and authentication via the authentication blockchain.
- Secure storage and management of password records in the data blockchain.
- Decentralized network with consensus mechanisms for authentication and data transactions.
- User-friendly command-line interface for easy interaction.

## Prerequisites

- Java Development Kit (JDK) 8 or higher.
- Network access for decentralized functionality.

## Installation

1. Clone this repository to your local machine:
   ```sh
   git clone https://github.com/yourusername/password-manager.git
   ```

2. Build the project using your preferred Java build tool (e.g., Maven or Gradle).

## Usage

1. Run the application:
   ```sh
   java -jar password-manager.jar
   ```

2. Follow the command-line prompts to register, log in, and manage your password records.

## Configuration

Configuration settings can be adjusted in the `config.properties` file. You may need to specify network parameters, encryption keys, and other settings.
